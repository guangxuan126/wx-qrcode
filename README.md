# wx-qrcode
微信小程序-二维码生成器

---

使用这个小程序可以轻松生成二维码
![](https://images.gxuann.cn/github/intro.png)

输入任何字符即可转换，注意**不超过255个字符**.<br>
这个小程序基于微信运行，请下载最新版本的微信，然后在`微信-发现-小程序`中打开.<br>
或者保存下面这个二维码至相册中，然后在微信中扫一扫即可打开.<br>
![](https://images.gxuann.cn/github/qr.jpeg)
